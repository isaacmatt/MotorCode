# MotorCode
Arduino and C++ code for control of 9V worm gear motors using Arduino Microcontrollers.

Work on this code was done as part of the University of Manitoba's Wind Energy Design Team (WEDESIGN) design for independent controllable motors in a small scale wind turbine.
Work was done between November 2023-June 2025.
